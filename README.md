# GameCogs, a marketplace/discography for used video games

GameCogs is a marketplace for used games, inspired by the popular music discography site Discogs.

## How it works (Theoretically)

Users can catalogue their video game collection, put games up for sale, and purchase games from other users.

Uses the Twitch and International Games Database (IGDB) API to fetch data on games.
